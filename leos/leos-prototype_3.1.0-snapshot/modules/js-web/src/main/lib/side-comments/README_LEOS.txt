This is a LEOS customized version of SideComments.js

The original open source code repository is available at:
* https://github.com/aroc/side-comments

The latest release (version 0.0.3) was published on 2014-07-03.
The master branch had several commits since the latest release.
The GitHub project has been marked as unmaintained by the author.

LEOS customization:
* code base taken from master branch with the latest commit on 2014-12-21.
* conversion of CommonJS module format into AMD module format (RequireJS).
* fixes to make code compatible with changes introduced in lodash v4.0.0.
* added user's department information to comments (JS and HTML templates).
* added event namespace and proper bind/unbind on creation/destruction.
